# PlacementCell-WebApp
Interface for the Employees of this company to fill in the data into the database and then download it in CSV
format.

## Tools Used 
- NodeJS
- MongoDB
- ExpressJS
- EJS
- Bootstrap

### Libraries: 
* axios
* bcryptjs
* connect-mongo
* cookie-parser
* dotenv
* ejs
* express
* express-ejs-layout
* express-session
* mongoose
* passport
* passport-local

## Getting started

* Clone the repository in your local system
* Install all required packages
```bash
npm install 
```
* Run project 
```bash
npm start
```
